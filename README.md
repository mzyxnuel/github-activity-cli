<center>
   <h3>
      Github activity CLI
   </h3>
</center>

Run for help:
``` bash
   github-activity -h
```

Options:
``` bash
SYNOPSIS
       github-activity [--username String] --type String

OPTIONS
       --username String
       GitHub username
       [Mandatory]

       --type String
       Type of activity: events, starred, or both
       [Optional, default = both]
```

Example: 
``` bash
github-activity [username] starred
```
